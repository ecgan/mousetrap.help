Peanut Butter with Jelly

For testing mousetrap with the Help extension.

Press ? to show help in a lightbox popup.

Code:

        Mousetrap.bind("p", function () { alert("Peanut butter is nice."); }, "alert peanut butter.");
        Mousetrap.bind("! j", function () { alert("Even better with Jelly."); }, "alert jelly.");
